# Cairog

Cairo binding for GNU Guile.
Documentation available at <https://zictogo.github.io/cairog/>.

## Installation

```bash
guix package -f guix.scm
info cairog
```

## Development

```bash
guix shell -D -f guix.scm               # development shell with all build inputs
./bootstrap && ./configure && make      # build in-tree
info -f doc/cairog.info                 # browse the manual without installing
```

## Tests

```bash
make check                              # on failure, check test-suite.log or tests/<name>.log
```

## Cleaning

```bash
make clean               # remove build artefacts
make distclean           # also remove files generated by ./configure
make maintainer-clean    # also remove files generated by ./bootstrap
```
